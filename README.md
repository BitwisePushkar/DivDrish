# DivyaDrishti

A simple app to help people distinguish between deepfake and AI-generated content.
Built with Flask, using Celery for background tasks, Redis for queuing, and PostgreSQL for storage.

## Features

* Deepfake & AI content detection
* Asynchronous processing
* Secure authentication

## Tech Stack

Flask · Celery · Redis · PostgreSQL · Docker

## Run with Docker

```bash
docker-compose up -d --build
```
